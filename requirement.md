# 피카츄 배구 Web 프로젝트 requirement

## ## 목적
* 과거 인기를 끌었던 '피카츄 배구' 게임을 현대적인 웹 기술로 재해석하여 구현합니다.
* GitHub Codespaces 환경에서 Python Flask 서버를 구동하고, 클라이언트(웹 브라우저)의 HTML5 Canvas와 JavaScript를 이용해 로컬 2인용 플레이가 가능한 웹 게임을 구축하는 것을 목적으로 합니다.

---

## ## 필수 기능
1. **게임 진입 및 초기화**
   * 게임 접속 시 '게임 시작' 버튼이 포함된 초기 화면(메인 메뉴) 표시
   * 5점 선취 시 전체 게임이 종료되며, 스코어 초기화 후 다시 초기 화면으로 이동
2. **로컬 2인용 플레이 제어**
   * 하나의 키보드로 두 명의 플레이어가 동시에 독립적으로 조작 가능
3. **2D 물리 및 충돌 감지**
   * 플레이어와 공, 네트와 공, 화면 경계(벽, 바닥, 천장) 간의 충돌 처리
   * 중력이 적용된 자연스러운 포물선 운동 및 점프 구현
4. **점수 시스템 및 게임 루프**
   * 공이 바닥에 닿았을 때 실점 영역을 판정하여 상대방에게 1점 부여
   * 실시간 점수(Score) UI 표시 및 승리 조건(5점) 감지

---

## ## 세부 기능

### 1. 조작 및 이동 시스템
* **1번 플레이어 (우측 캐릭터)**
  * `←`, `→` (좌우 이동), `↑` (점프)
  * `Enter` (스파이크)
* **2번 플레이어 (좌측 캐릭터)**
  * `A`, `D` (좌우 이동), `W` (점프)
  * `Z` (스파이크)
* **연직 운동 (점프 및 낙하)**
  * 중력 가속도를 적용하여 상승 후 자연스럽게 하강
  * 공중 상태에서는 다리 애니메이션이 정지(고정)됨

### 2. 애니메이션 시스템
* **이동 애니메이션**: 플레이어가 좌우로 이동할 때 2장의 다리 교차 이미지(프레임)가 번갈아 출력되는 워킹 애니메이션 구현
* **정지 및 공중 상태**: 이동하지 않거나 점프/낙하 중일 때는 기본 정지 프레임 유지

### 3. 공(Ball)의 물리 및 스파이크 메커니즘
* **초기 서브**: 게임 시작 또는 득점 후, 양쪽 진영 중 한 곳의 공중에서 공이 자연스럽게 떨어지며 시작
* **충돌 반사**: 공이 캐릭터 몸에 부딪힌 방향/각도에 따라 물리적인 반사각 반영
* **네트 물리**: 화면 중앙에 위치한 네트(전체 화면 높이의 1/3)에 공이 부딪히면, 통과하지 못하고 네트에 걸려 아래로 툭 떨어지는 물리 효과 구현
* **스파이크 시스템**: 공이 캐릭터에 닿아 있는 순간에 스파이크 키가 눌려 있으면 발동 (공의 속력이 급격히 빨라짐)
  * **좌측 플레이어 스파이크**: 
    * `W`(위) 키와 함께 스파이크: 오른쪽 위 대각선 방향으로 강하게 발사
    * `D`(오른쪽) 키와 함께 스파이크: 직선에 가까운 빠른 포물선으로 오른쪽 발사
  * **우측 플레이어 스파이크**: 
    * `↑`(위) 키와 함께 스파이크: 왼쪽 위 대각선 방향으로 강하게 발사
    * `←`(왼쪽) 키와 함께 스파이크: 직선에 가까운 빠른 포물선으로 왼쪽 발사

### 4. 턴/라운드 종료 및 승리 조건
* **득점 판정**: 공이 네트를 기준으로 왼쪽 바닥에 닿으면 우측 플레이어 +1점, 오른쪽 바닥에 닿으면 좌측 플레이어 +1점 (득점하지 않은 플레이어의 점수는 유지)
* **게임 종료**: 어느 한 쪽이라도 5점에 먼저 도달하면 게임 세트 종료
* **초기화**: 게임 종료 즉시 양쪽 스코어는 0점으로 리셋되며 메인 초기 화면으로 강제 이동

## ## 디자인 요구
* **화면 구성**: 중앙에 화면 높이의 1/3을 차지하는 수직 네트 배치
* **그래픽 에셋**: 별도의 외부 리소스가 준비되지 않았으므로, 개발 초기에는 HTML5 Canvas 내부 그래픽 기능을 활용하여 구현
  * 플레이어 캐릭터: 피카츄 형태를 단순화한 사각형 또는 라운드 형태의 도형 (이동 시 하단 다리 표현 영역 교차 변화)
  * 배구공 및 네트: 식별이 명확한 원형 및 직사각형 색상 도형 활용
* **UI**: 화면 상단 중앙에 현재 스코어(Player 1 vs Player 2)를 직관적이고 큼직하게 표시

## ## 기술 스택
* **Backend (Server)**: Python Flask (GitHub Codespaces 환경에서 빌드 및 웹 서빙 역할 담당)
* **Frontend (Client)**: HTML5 Canvas, CSS3, Vanilla JavaScript (웹 게임 핵심 로직, 렌더링 및 키보드 이벤트 핸들링 전체 담당)
* **Development Environment**: GitHub Codespaces, Git
